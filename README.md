## hardhat框架

开发 编译 测试 部署


```bash

npm i hardhat

npx hardhat
# 编译
npx hardhat compile
# clean
npx hardhat clean

npx hardhat run script/deploy.ts
# 启动一个区块链网络 服务节点默认的json-rpc地址是http://127.0.0.1:8545  同时生成账号
npx hardhat node

```
